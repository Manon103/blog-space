# blog-space

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```

## Current Step

- 第一阶段：静态网页

  第一阶段为纯静态网页，博客内容存储在前台，并通过懒加载获取文章资源

- 第二阶段：node.js + mongoDB
